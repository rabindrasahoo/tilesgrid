INTRODUCTION
------------

This module provides Grid View of content of Tile Content type.


REQUIREMENTS
------------

This module requires 

	-Views (core)
	- colorbox
  	- colorbox_simple_load
 	- entityqueue

INSTALLATION
------------

1.	Install the module and it’s dependencies.
2.	Add contents to Tile and Article content type. Add terms to Tags taxonomy. 
3.	Add Entity queue items to "Sorting Tiles" type.
4.	Enable Block "Grid Tile" to any region to view the Grid Tiles.
